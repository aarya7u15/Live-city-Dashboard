MyCity – Smart City Dashboard

MyCity is a simple React-based web app that shows useful information about a city in one place. Instead of checking different apps/websites, users can select a city and view things like weather, food options, and latest alerts together.

Idea -
The main idea behind this project is to build a clean dashboard where users can quickly get important updates about a city. It uses real APIs to fetch live data and display it in a structured way using React.

Features -
- Weather information of the selected city
- Food and restaurant suggestions
- Latest alerts / news related to the city
- Basic transport / traffic-related information  

Tech Stack -
- React (Functional Components)
- JavaScript (ES6)
- Tailwind CSS
- Fetch API
